**Raindrop**

Mein Abschlussprojekt für den Programmieren 1. Workshop in Lüdenscheid. 

*Was kannst du damit machen*
Kurz gesagt garnichts. Es ist eine einfache Animation in welcher "Regentropfen" über den Bildschirm wandern. 
In einer im Code Festgelegten Zeit (in ms) füllt sich der Bildschirm mit einer Farbe (Blau).

*Die UI*
Ich habe passend zum Regen-Thema auch eine Regen-UI für turtle.js programmiert. Diese ist in css geschrieben und auch individuell anpassbar.
